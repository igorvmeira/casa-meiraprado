⚙️ Configurar Firebase

Cole abaixo as configurações do seu projeto Firebase. Esses dados ficam salvos no navegador. Você só precisa fazer isso uma vez.

Como obter: Firebase Console → Configurações do Projeto → Seus apps → SDK do Firebase.

Dica: No Firebase, vá em Firestore Database → Regras e coloque allow read, write: if true; para acesso público (qualquer pessoa com o link pode editar).
